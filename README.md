# semantic_vue_ui

依赖于semantic-ui中的样式来开发的一套vue的公共组件。
仅供学习使用哈。


# 安装

安装两个包：

- demo-ui-93
- semantic-ui-css

```
npm i semantic_vue_ui
```

# 引入

在main.js中

```

// 引入组件
import UI from 'semantic_vue_ui'
// 引入样式
import "semantic_vue_ui/lib/semantic_vue_ui.css"

Vue.use(UI)
```

# 使用

##  my-headline

```
<my-headline>标题</my-headline>
```
